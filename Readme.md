### 软件包内容

\1. Pad_Firmware_Modify_Tool_V1.15/**PAD固件修改工具**用于修改ROM包
\2. Card_Product_Tool_V2.00/**量产卡固件烧写工具**用于把ROM包烧录到SD卡中
\3. Pad_Product_Tool_1.09/**PAD量产工具**用于用USB把ROM包烧录到PAD中



另有SmartBook刷机ROM分享，修改内容如下：

1.修改开机画面为RUNN
2.删除所有预装应用，保留Google Play服务
3.预置启动器更改为WINDOWS MOBILE
4.修改默认语言为中文
5.修改型号为RUNN S500

6.预置如下系统APP
（1）应用中心HD
（2）云备份HD
（3）Adobe Flash Player
（4）SuperUser

7.预置如下用户APP
（1）QQ HD
（2）微信
（3）UC HD
（4）WPS Office
（5）QQ音乐HD
（6）爱奇艺HD

8.第一次开机时间较长，请耐心等待，开机需要做的操作
（1）选择默认桌面为魔伴Windows桌面，始终
（2）左键长按桌面，图标大小选择为小图标
